Configurações

Segurança, exportação e integrações

Identidade visual
Padrão

Substitua o ícone padrão pelo seu logo. A imagem será exibida na barra lateral e na tela de acesso. Recomendado: PNG ou SVG quadrado, fundo transparente.

Logo atual
Ícone padrão do Iryx
A imagem fica salva localmente neste dispositivo. Para aparecer em todos os dispositivos, repita o upload em cada um.
Face ID / Touch ID
Não configurado

Protege o acesso ao Iryx com biometria. Suas credenciais do Supabase ficam criptografadas e só são liberadas após autenticação.

Exportar dados

Exporte suas transações para CSV (Excel) ou PDF. Útil para declaração de IR ou enviar ao contador.

Notificações
Desativadas

Alertas quando um orçamento estiver próximo do limite. Requer Safari iOS 16.4+ com o app salvo na tela de início.

Para funcionar no iPhone: abra o Iryx no Safari → compartilhar → "Adicionar à Tela de Início" → abra pelo ícone criado → ative notificações aqui.
Lembretes de calendário

Crie eventos no seu Calendário para contas fixas recorrentes. O arquivo .ics abre direto no Calendário do iPhone ou Mac.

Aparência

O tema claro/escuro segue automaticamente as configurações do seu dispositivo.

Conexão Supabase
Conectado

Use este botão em um novo dispositivo ou se as credenciais precisarem ser atualizadas. As suas transações no Supabase não serão apagadas.
